Scores

Lorsque tu ajoutes un jeu à la base de données, il est important de spécifier les scores minimum et maximum. Ces valeurs définissent les limites du score que les joueurs peuvent obtenir dans ce jeu.

Le score minimum représente le niveau le plus bas de performance attendu dans le jeu, tandis que le score maximum représente le niveau de performance le plus élevé qu'un joueur peut atteindre.

Le score d'un joueur est calculé comme suit :

score = ( score_individuel - score_min score_max - score_min ) × 1000
